## üíæ Installation

### Requirements
- Node.js 20+
- TypeScript 5.9+

```bash
# Using pnpm (recommended)
pnpm add @tuwaio/orbit-evm @wagmi/core viem
# Using npm
npm install @tuwaio/orbit-evm @wagmi/core viem
# Using yarn
yarn add @tuwaio/orbit-evm @wagmi/core viem
```
